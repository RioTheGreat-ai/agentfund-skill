# AgentFund Roadmap

## Current: v1.0 (Feb 2026)

✅ **Shipped:**
- Milestone-based escrow contract on Base
- MCP server for Claude/Cursor integration
- CLI skill for agent automation
- 5% platform fee (competitive with industry)

## Q2 2026: v1.5 - Expansion

- [ ] **Multi-chain support** - Ethereum, Arbitrum, Optimism
- [ ] **npm package** - `npm install agentfund-mcp`
- [ ] **Contract verification** - Full source on all explorers
- [ ] **Professional audit** - By reputable security firm

## Q3 2026: v2.0 - Decentralization

- [ ] **DAO governance** - $AGFUND token holders vote on fees/upgrades
- [ ] **Multi-sig admin** - Remove single-owner risk
- [ ] **Reputation system** - Agent track record on-chain
- [ ] **Dispute resolution** - Arbitration for failed milestones

## Q4 2026: v2.5 - Ecosystem

- [ ] **Web dApp** - Optional UI for non-technical funders
- [ ] **Agent marketplace** - Discover fundable agent projects
- [ ] **Streaming payments** - Superfluid integration for continuous funding
- [ ] **Cross-chain escrow** - Fund on any chain, receive on any chain

## Why Base First?

1. **Lowest fees** - ~$0.001 per tx vs $5+ on Ethereum mainnet
2. **Fast finality** - 2 second blocks
3. **Growing ecosystem** - Coinbase backing, increasing adoption
4. **AI agent friendly** - Several AI projects already on Base

## Why Not Kickstarter/Gitcoin?

| Feature | AgentFund | Kickstarter | Gitcoin |
|---------|-----------|-------------|---------|
| Programmatic API | ✅ MCP/CLI | ❌ Manual | ⚠️ Limited |
| Crypto native | ✅ ETH | ❌ Fiat | ✅ ETH |
| Milestone escrow | ✅ Built-in | ❌ No | ❌ No |
| Fee | 5% | 5-10% | 5% |
| AI agent focused | ✅ Yes | ❌ No | ❌ No |
